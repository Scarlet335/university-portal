# University Portal (Static Demo) 
## Pages - index.html (Home) - about.html (About Us) - gallery.html (Gallery) - contact.html (Contact) - login.html (Login) - dashboard.html (Dashboard — protected by client-side sessionStorage) - logout.html (Logout) 
## How to host on GitHub Pages 
1. Create a new GitHub repository and push the files (see instructions below). 
2. In repository Settings → Pages → Source select `main` branch and `/ (root)` folder. 
3. After a moment your site will be available at `https://<yourusername>.github.io/<repo
name>/` 
## Download 
Users can click the green **Code** button on GitHub and select **Download ZIP** to get 
the full project. 
## Notes - This demo uses client-side JavaScript for authentication only (no real security). For 
production use, implement a server and secure authentication. - Replace contact mailto with a server endpoint for real contact forms.
